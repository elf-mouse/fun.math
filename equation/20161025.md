**Q:** 学生问老师几岁了，老师回答，当我是你现在的年龄时，你只有 3 岁。当你是我现在的年龄时，我已经 63 岁，问老师和学生几岁？

**A:**

```
求
teacher: X
student: Y

条件1
teacher: Y
student: 3

条件2
student: X
teacher: 63

解
∵ X - Y = Y - 3
∴ X = 2Y - 3

∵ X - Y = 63 - X
∴ Y = 2X - 63

方法1
∵ X = 2(2X - 63) - 3

∴ X = 4X - 126 - 3
∴ 3X = 129
∴ X = 43

∵ Y = 2X - 63
∴ Y = 86 - 63 = 23

方法2
∵ Y = 2(2Y - 3) - 63

∴ Y = 4Y - 6 - 63
∴ 3Y = 69
∴ Y = 23

∵ X = 2Y - 3
∴ X = 46 - 3 = 43

答
teacher: 43
student: 23
```
